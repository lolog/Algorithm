1.线性表
  L = (a1, a2, a3, ...., an)
  Describe:
    L  - 表名
    ai - 表中的元素,亦或结点、表项
    n  - 表项的个数,亦或表的长度
  Attach:
    n=0  - 空表
    a1   - 表头(head)
    an   - 表尾(tail)
  Append:
    L - 有限序列,有唯一的表头和表尾,表中所有的数据元素都具有相同的数据类型。
    Sorted List  (有序线性表) - 元素按照值的递增顺序排列
    Unsorted List(无序线性表) - 元素的值与位置之间没有特殊的联系
  Category:
    (01) 顺序表
    (02) 链表
